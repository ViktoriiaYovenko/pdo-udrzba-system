# Návrh systému podpory údržby a provozního plánování pro sociální výrobní organizaci

## Popis projektu
Projekt představuje návrh jednoduchého informačního systému pro evidenci poruch výrobních zařízení a podporu rozhodování v oblasti údržby a plánování provozu. Systém je navržen pro malou sociální organizaci zaměřenou na ruční výrobu (šití, práce se dřevem, výtvarná činnost).

Součástí návrhu je analytický modul, který na základě historických dat vyhodnocuje četnost poruch a doporučuje preventivní zásahy. Systém rovněž umožňuje zohlednit externí faktory, například vysoké teploty v letním období.

## Cílová skupina
- Vedoucí organizace
- Správce provozu
- Údržbář
- Administrativní pracovník

## Typ dokumentace
- Concept dokumentace (architektura systému, logika analýzy, návrh řešení)
- Procedurální dokumentace (návody k používání systému)
- Reference dokumentace (datový model, pravidla vyhodnocování, algoritmus prioritizace)

## Cíl práce
Cílem projektu je navrhnout systém, který:
- Zlepší přehled o poruchách zařízení
- Podpoří preventivní údržbu
- Sníží závislost na individuální zkušenosti pracovníků
- Pomůže efektivně plánovat provoz v závislosti na provozních a klimatických podmínkách
